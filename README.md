## MovieMaster: A MERN-based project

## Live URL: https://moviemaster-820a2.web.app/

## Theme:

A comprehensive movie management system where users can browse, manage, and
organize their favorite movies with advanced filtering and personal collections.

## Header:

Initially, it contains Home, All Movies, My Collection, Login & Register menu. A logged-in user see some additional features in menu, he/she can access watchlist, and add movie feature. Also clicking the user image icon shows a dropdown.

## HomeLayout:

It includes statistics, recently added movies, top-rated movies, About MovieMaster Section and an animation.

Key Features

- **Secure User Authentication:** Easily register and log in for a personalized movie experience.
- **Personal Feature:** Add, update, add to watch list and delete movies feature for logged-in users.

-**Filter Option:** Allows to filter movies according to genres and ratings.

- **Fully Responsive Design:** Provides seamless experience on desktops, tablets, and mobile devices.
- **Theme Toggle:** Switch between dark and light modes for comfortable viewing.
